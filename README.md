# 信用卡欺诈检测（分类项目）

## 项目目标
基于极度不平衡的信用卡交易数据，构建分类模型识别欺诈交易。

## 数据特点
- 样本量：284,807 条
- 特征：30 个（PCA 变换）
- 不平衡程度：欺诈类仅占 0.17%

## 使用模型
- 随机森林（Random Forest）
- 逻辑回归（Logistic Regression，用于对比）

## 评估方式
- 不使用准确率（会被不平衡主导）
- 主要指标：少数类（欺诈）的 **F1、召回率、AUC**

## 核心结论
- 随机森林在少数类上的 F1 显著优于逻辑回归
- 原因：欺诈数据非线性 + 强不平衡，线性边界不适用
- 混淆矩阵显示：漏报 29 笔欺诈，误报 5 笔正常交易。
- AUC 为 0.97，说明模型排序能力较强，可在有限审查资源下优先定位高风险交易。

- ## 运行环境
- - pandas / scikit-learn / matplotlib
